Это учебный проект Хекслет. Его задача, реализовать несколько игр

## System Requirements

* NodeJS >= 18.15.0
* Make >= 4.3

## Install

1. `git clone git@github.com:Nemial/frontend-project-44.git`
2. `make install`
3. `npm link`
4. Done
